Task Management.

1. Installed Laravel Project.
2. Setup Laravel 8.0
3. Composer Update
3. NPM install and NPM run dev.
4. run on http://localhost/task




after run project
 1. click on create task
 2. enter task name and priorities
 3. redirect to list page
 4. edit or delete task.
